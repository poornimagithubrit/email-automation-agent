# 📧 Email Automation Agent

> An agentic AI workflow that automates end-to-end email processing — parsing incoming emails, validating attachments and proof URLs, routing multi-step approvals, and sending structured responses automatically.

---

## 🧩 Problem

Teams handling high volumes of transactional emails (invoices, approvals, reconciliations) spend hours manually reading, validating, and forwarding emails to the right people. Delays cause bottlenecks and compliance issues.

## ✅ Solution

An intelligent email agent that:
1. **Reads and classifies** incoming emails automatically
2. **Extracts key data** (amounts, dates, references) from email body and attachments
3. **Validates proof URLs** embedded in emails
4. **Routes to approval stages** based on business rules
5. **Sends automated replies** with structured summaries

---

## 🏗️ Architecture

```
Incoming Email (IMAP/Gmail API)
        │
        ▼
  Email Parser & Classifier
        │
        ▼
  LangChain Agent
  (Multi-step Reasoning + Tool Use)
        │
        ├──► Attachment Extractor (OCR/PDF parsing)
        │
        ├──► Proof URL Validator
        │
        ├──► Approval Router (Stage 1 → 2 → 3)
        │
        └──► Auto Reply Generator (GPT-4o)
                  │
                  ▼
            SMTP / Gmail API (Send Response)
```

## ⚙️ Setup

```bash
git clone https://github.com/poornimagithubrit/email-automation-agent
cd email-automation-agent
pip install -r requirements.txt
cp .env.example .env
# Add your API keys and Gmail credentials
```

## 🚀 Usage

```bash

uvicorn api.main:app --reload

# Trigger email processing manually
POST /process-emails

# Check approval status
GET /approvals/{email_id}
```
